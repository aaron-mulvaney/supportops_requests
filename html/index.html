Support Ops Requests

There are multiple types of Requests that Support Ops manage, see below for more information:


Internal Service Requests (ISR)

ISR's capture many different types of "Issues"

Bug

Data Purge

Data Fix/Update

Data Extract

App Integration Issue

Security

Problem Management

Custom Work

Support Ops Project (SOP)

Maintenance Change Request (MCR)

Both of these types of request are logged as stories.


You can log a request for Support Ops using the following link:

https://jira.hmhco.com/secure/CreateIssue!default.jspa
